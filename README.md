# sample-websocket
WebSocketを体感するためにWebSocketを用いた簡単な双方向通信アプリケーションを作成

## Execution
### Server
```
$ node app.js
```

### Client
```
# ターミナルを複数開いて実行してみる
$ node client.js
```
